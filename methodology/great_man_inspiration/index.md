## Naming convention

Use stable topic-first filenames inside the theme subfolders:
- `001-on-protracted-war-long-cycle-embedded-software.md`
- `003-red-political-power-foothold-thinking.md`
- `000-mao-anthology-sequential-reading-overview.md`

Rules:
- `000` for overviews and corpus-level summaries
- sequential numbers for article-based notes inside a series
- no date prefix in the main filename when the thematic order already matters
- keep one note per file
