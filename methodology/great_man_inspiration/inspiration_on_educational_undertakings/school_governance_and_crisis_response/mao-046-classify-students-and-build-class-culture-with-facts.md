# 046《团结一切抗日力量，反对反共顽固派》对教育从业者的启发：对学生分类不搞扩大化，用"十没有"立班级文化

## 原文出处
- 毛泽东《团结一切抗日力量，反对反共顽固派》，1940 年 2 月 1 日
- 原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19400201.htm

## 原文核心观点（为初中政治老师提炼）
1. 气愤是正当的，但**不能扩大化**，对不同学生应采取不同政策。
2. **两条政策并行**：坚决处理违纪 + 积极扶持大多数，同时做。
3. 用**可验证的具体事实清单**（"十没有"）建立班级/学校的文化证据。
4. 真整合 vs 假整合：以消灭学生个性为代价的"整齐划一"是假的管理。

## 转译到教育从业者的启发

### 启发一：别一棒子打翻一个班——精细分类是教育的底线

毛原文警示：不能因为几个坏蛋就把整个群体都否定。
对老师最贴切的场景——**班级出现违纪/集体事件时的处理态度**。

典型错误场景：

- 班级某几个男生起哄上课 → "这个班就是没救" → 对全班冷脸
- 几个学生作业抄袭被发现 → "这届学生品性有问题" → 不再信任任何作业
- 某个学生顶撞老师 → "初中生都不懂事" → 对所有学生戒备
- 发现一个学生偷东西 → "这个班风气差" → 班会拿全班开涮
- 家长群几个家长闹事 → "这帮家长都难搞" → 对所有家长摆谱

**扩大化的伤害极大**——你把情感绑在坏的少数人身上，**伤害了大多数**。

**毛的方法论对老师**：
分类是教育的底线——**精细识别**哪几个学生是核心问题、哪些是跟随、哪些是旁观、哪些是无辜：

| 类别 | 识别 | 处理 |
|---|---|---|
| 主要违纪者 | 具体指向的 2-3 人 | 坚决严肃处理，家长沟通，班主任联动 |
| 跟随者 | 被带动的 3-5 人 | 单独谈话，讲清楚后果，给机会 |
| 旁观沉默者 | 多数 | 肯定他们的克制，邀请未来做正向力量 |
| 主动制止者 | 少数 | 当众表扬，树立正向榜样 |

**一句话**：
对事件严厉 ≠ 对全班严厉。
你对全班施压，会把第 3 类和第 4 类学生也推到对立面。

### 启发二：两条政策并行——严管违纪 + 积极扶持

毛："一方面反对……一方面团结……"

老师最容易犯的单腿错误：

**只会严厉的老师**：
- 天天批评 → 学生恐惧，不敢接触
- 只抓问题 → 学生觉得老师不喜欢自己
- 政治课变成训诫课 → 学生不听

**只会温柔的老师**：
- 什么都不管 → 违纪学生越来越过分
- 怕得罪学生 → 班级底线崩塌
- 政治课变成说笑课 → 学生不尊重

**"两条政策并行"对老师**：

**"严管"政策（对违纪）**：
- 违反红线当场处理，不留"下次"
- 考试作弊零容忍
- 欺凌事件第一时间介入
- 严重违纪必须家长+班主任三方谈

**"扶持"政策（对大多数）**：
- 每周至少 3 名学生单独鼓励（换着来，不要只夸少数人）
- 每次作业找出闪光点批注
- 课前 5 分钟常设"本周我的进步"环节
- 生日祝福/小奖励等仪式感
- 对家长主动反馈孩子的闪光点（不要只在出事时联系家长）

**两条腿同时走**，学生才会**既敬畏又亲近**。
只会严的老师学生怕你不爱你；只会柔的老师学生不怕你也不尊重你。

### 启发三：用"十没有/十有"立班级文化

毛的"边区十没有"对老师最直接的启发——
**班级文化不是喊口号，是列具体事实**。

老师对班级文化的描述常见毛病：
- "我们班同学之间团结" ← 空
- "我们班学风很好" ← 空
- "我们班有正能量" ← 空

学毛的做法——**和学生一起写"我们班的十没有/十有"**：

**范例（可贴教室墙上）**：

**我们班的十没有**：
1. 没有同学被起外号嘲笑
2. 没有人因成绩被孤立
3. 没有考试作弊
4. 没有欺凌和孤立
5. 没有抄作业
6. 没有上课拿手机
7. 没有用暴力解决矛盾
8. 没有分派别小团体
9. 没有背后说同学坏话
10. 没有迟到超过 5 分钟

**我们班的十有**：
1. 有同学困难互相帮助的传统
2. 有每周值日轮换的自治
3. 有午休安静不吵闹的约定
4. 有"进步之星"每月评选
5. 有主动整理教室的自觉
6. 有早读声音洪亮的气势
7. 有课前 1 分钟安静就位的习惯
8. 有对卫生扫把放回原处的规矩
9. 有尊重每一位代课老师的礼貌
10. 有班干部轮换的民主传统

**这些不是口号，是可现场查证的具体事实**。

**做法**：
- 学期初和学生一起列清单（让他们投票）
- 每月班会一条条打勾：做到还是没做到
- 没做到的想办法改进
- 学期末公开"我们做到了 X/10 条"

**价值**：
- 让学生理解"文化 = 可验证的事实"，不是喊的
- 培养他们**写出具体事实而不是形容词**的思维习惯
- 这个思维习惯是真正的政治课核心能力

### 启发四：真整合 vs 假整合——别让"整齐划一"毁了学生的个性

毛反对"以消灭对方为代价的统一"——
在教育语境里对应**"班级管理不是把学生都捏成一个模子"**。

老师/学校常见的"假整合"：
- **"你们要统一思想"** → 强压不同声音
- **"好学生就应该是 XX 样子"** → 消灭性格差异
- **"我们班要团结" → 实际上是排挤个性强的学生**
- **"家长都要配合老师"** → 不听家长合理反馈

毛的三问对教育同样适用：
1. 是否有真正的共同目标？（学习成长 vs 老师管理省心）
2. 是否尊重各自独立性？（学生性格/兴趣/家庭差异）
3. 是否各方都在成长？（老师也在成长？学生也在成长？家长也在成长？）

**真正的好班级文化**的特征：
- **底线高度统一**（违反红线必处理）
- **上层充分多元**（性格/兴趣/表达方式可以不同）

这就像毛说的——
**统一应该是"抗战、团结、进步"三件事的统一**，
不是把所有人都捏成一个样子。

对政治老师特别重要：你教的就是**如何在多元中建立共识**这门课。
如果班级文化本身是"压抑个性"的，那你教的政治课内容和你做的班级管理就矛盾了。

## 给自己的三个自检问题
1. 我最近因为班级几个学生的问题，有没有对全班表达过失望/冷漠？大多数学生是不是被误伤了？
2. 我最近一个月有没有同时做"两条政策"——对违纪坚决处理 + 对大多数学生主动积极关怀？还是只会一条？
3. 我们班的"文化"能不能写成"十没有 + 十有"的可验证清单？如果写不出来，说明"文化"还是口号。
