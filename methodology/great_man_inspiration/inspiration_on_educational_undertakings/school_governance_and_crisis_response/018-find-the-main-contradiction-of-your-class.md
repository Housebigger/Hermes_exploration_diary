# 018 — 抓住班级的主要矛盾，别同时跟所有问题开战

原文对应：
《矛盾论》
https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19370801.htm

写作背景：
1937年，毛泽东系统阐述了矛盾的普遍性与特殊性，尤其强调：在复杂局面中，必须找到"主要矛盾"和矛盾的"主要方面"，集中力量解决它，其余矛盾才会随之松动。

说明：
以下转译借鉴核心方法——**每个班级都有一堆问题，但决定班级走向的往往只有一两个核心矛盾，找到它比什么都重要**。

## 核心思想的教育场景转译

### 班级里的"矛盾清单"永远很长

纪律差、成绩低、班风散、个别学生刺头、家长不配合、同事不协作、学校任务压人、课时不够……

如果你试图同时解决所有问题，结果就是什么都没解决。

### 教育从业者的实操启发

1. **诊断"主要矛盾"：到底是什么卡住了这个班级？**
   - 有些班级的主要矛盾是"核心刺头带坏了风气"→ 集中精力转化关键人物
   - 有些是"学生完全没有学习方法"→ 花时间教学习方法比讲内容更重要
   - 有些是"班级没有凝聚力，同学关系冷漠"→ 先做团建，再谈学习
   - 有些是"家校关系崩了"→ 先修复信任，再要求配合

2. **主要矛盾解决了，次要矛盾会跟着缓解**
   如果班级纪律差的根源是三个核心学生在带节奏，那你把这三个人的问题解决了，全班纪律可能自然好转。不需要每个人都盯。

3. **矛盾是会转化的——上学期的主要矛盾可能不是这学期的**
   开学初主要矛盾可能是"规则还没建立"，期中可能变成"学习分化加剧"，期末可能是"考试焦虑"。跟着节奏调整重心。

4. **矛盾的"主要方面"：问题出在老师还是学生？**
   有时候班级问题的主要方面不在学生，而在老师自己的方法或态度。如果你换了三套管理方案都不管用，也许该反思的不是方案，而是你跟学生之间的关系本身。
