# 045《克服投降危险，力争时局好转》对股市投资的启发：牛熊共存时的双管齐下

## 原文出处
- 毛泽东《克服投降危险，力争时局好转》，1940 年 1 月 28 日
- 原文链接：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-19400128.htm

## 原文核心观点（为投资者提炼）
1. **两种方向同时存在，两种前途都有可能**——投降方向和抗战方向同时在斗争中。
2. 不要把各地发生的严重现象**孤立**起来看，要识别它们是同一个趋势的不同表现。
3. 识别"**有利条件**"和"**不利条件**"两张清单，双管齐下。
4. 在保持积极进攻的同时，**提起突然事变的警觉性**。

## 转译到股市投资的启发

### 启发一：牛熊共存——同一时点两种前途

毛说："两种方向同时存在，两种前途都有可能。"
这句话对投资者的意义：**任何一个时点，市场都既有向上的力量也有向下的力量**，
不存在"现在纯粹是牛市所以只看多"或"现在纯粹是熊市所以只看空"。

典型场景：

- **牛市里的风险累积**
  - 估值拉伸到历史高位 → 向下的种子已种下
  - 杠杆资金持续进场 → 流动性风险上升
  - 基金大量发行 → 筹码分布在散户手上（弱手）
  - 此时多数人在喊"慢牛刚开始"
- **熊市里的机会孕育**
  - 龙头股估值跌到历史低位 → 向上的种子已种下
  - 股权回购/大股东增持频出 → 产业资本看多
  - 融资盘强平结束 → 卖盘枯竭
  - 此时多数人在喊"还没见底"

**关键误区**：大多数散户在牛市里只盯"抗战方向"，
在熊市里只盯"投降方向"——这就是毛批评的**孤立化、单向化思维**。

**口袋里的一句话**：
不管你当下判断是牛是熊，强迫自己同时列两张表——
一张"如果我对了，后续会看到哪些信号"，
一张"如果我错了，会从哪些地方先出问题"。

### 启发二：警惕"小回撤孤立看"的滑坡

毛说："不要把各地发生的……严重现象孤立起来看。"

投资里的翻版：**不要把多只持仓的小幅回撤孤立看待**。

典型的危险信号（单独看都不大，合起来是重大预警）：

- A 股跌 8%，你想"个股波动而已"
- B 股跌 5%，你想"这个板块最近都弱"
- C 股跌 3%，你想"正常调整"
- 同时指数跌 4%，你想"不深"
- 同时你持仓的 ETF 全线飘绿

如果这些**同时发生在一两周内**，这不是孤立事件，
这是**系统性的风险偏好切换**——一旦继续恶化就会连杀。

**识别工具**：账户层面的"体温计"。
每周五收盘做一次体检：
- 本周持仓里多少只创了 20 日新低？
- 本周账户总浮亏变化是否加速？
- 本周指数和你的持仓同步走弱？

连续 2-3 周这些指标恶化 → 不是孤立 → **可能需要减仓/止损**，
而不是继续安慰"每只都是独立原因"。

### 启发三：不要只从"怕踏空"或"怕套牢"一点做决策

毛说："如果只从惧怕破裂……一点设想，那末，抗战的前途就是危险的。"

投资者最大的单维度决策错误：

- **只从"怕踏空"一点出发** → 牛市后期追高
  - 每次看到别人赚钱就焦虑
  - 不管估值直接冲进去
  - 结果在最差位置接盘
- **只从"怕套牢"一点出发** → 熊市底部割肉
  - 每次浮亏就焦虑
  - 不管基本面直接砍仓
  - 结果在最差位置交出筹码
- **只从"怕波动"一点出发** → 永远只买理财，错过长期复利
- **只从"怕错过大牛股"一点出发** → 满仓押单票，赌上全部身家

毛的方案是"**双管齐下**"：进攻方向 + 防守方向同时设计。

翻译到投资：
- 账户里必须同时有"进攻仓"（博取超额收益）和"防守仓"（现金/低波动资产）
- 任何单边押注都要有对应的对冲或止损
- 决策时永远问两个问题：如果我对了会怎样？如果我错了会怎样？

**口袋里的一句话**：
你最近一笔操作，如果剥掉情绪只剩"怕 X"这一个原因，
它很可能是一笔错误决策。

### 启发四：永远列两张清单——有利条件 + 不利条件

毛在 1940 年那个时点，**同时列出**国际国内的有利条件和不利条件，
然后在此基础上决策。

投资者面对任何"要不要买"或"要不要卖"的决策时，应照搬这个方法：

```
【这只股票/这个市场的有利条件】
- 估值处在历史 X 分位
- 产业政策支持
- 基本面环比改善
- 机构持仓低位
- 行业出清近尾声

【这只股票/这个市场的不利条件】
- 宏观流动性收紧
- 行业竞争加剧
- 公司治理瑕疵
- 短期盘面弱
- 风格切换中
```

**决策原则**：
- 有利条件明显多于不利条件，且都是大权重因素 → 加仓
- 两者大致相当 → 维持
- 不利明显多于有利 → 减仓
- 只看得到一边（要么全利好要么全利空） → **先停手**，这说明你已经情绪化了

### 启发五：在积极进攻时保持"突然事变"警觉

毛说："力争时局好转，同时提起可能发生突然事变的警觉性。"

这句话翻译到投资：**仓位可以重，但必须有黑天鹅预案**。

具体动作：
- 仓位重的时候，**必须**有硬性止损位（不是"看情况"）
- 单票集中度高时，**必须**对该行业的重大风险事件做情景推演
- 使用杠杆时，**必须**预留 30% 以上的保证金缓冲
- 重大事件前（大选/议息/财报），**必须**评估是否需要降仓

**一句话**：敢于看多是一种能力，
但看多时不准备看空预案，就是赌徒。

## 给自己的三个自检问题
1. 我当前的仓位结构里，同时包含"看多逻辑"和"看空对冲"吗？还是单边押注？
2. 我最近的账户里，有几只票同时在阴跌？我把它们当独立事件看，还是识别出这是同一条风险敞口？
3. 我最近一笔决策，如果剥掉情绪只剩一个原因（怕踏空/怕套/怕错过），这个决策经得起推敲吗？
